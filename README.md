# Secure Login System

## Overview

A secure login web application developed using Flask, SQLite, and bcrypt.

## Features

* User Registration
* Secure Password Hashing using bcrypt
* Login Authentication
* Input Validation
* Protection against SQL Injection using parameterized queries
* Session Management
* Logout Functionality

## Technologies Used

* Python
* Flask
* SQLite
* bcrypt

## How to Run

pip install flask bcrypt

python app.py

## Security Features

* Password Hashing
* SQL Injection Prevention
* Session Management

## Expected Outcome

A secure login system that protects user credentials and prevents common authentication attacks.
